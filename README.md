Sitio web estático para **BiciExpress Quito**, un taller y servicio de alquiler de bicicletas en la ciudad de Quito.  
El proyecto presenta una estructura clara, navegación funcional y diseño visual moderno, ideal para mostrar los servicios de forma accesible y atractiva.

---

##  Estructura del proyecto
biciexpress-quito/ │── index.html │ ├── css/ │     └── style.css │ ├── img/ │     └── banner.jpeg │ └── pages/ ├── servicios.html ├── alquiler.html ├── acerca-de.html └── contacto.html
---

##  Contenido

- **index.html**  
  Página principal con logotipo ficticio, menú de navegación, imagen destacada, mensaje de bienvenida y descripción del servicio.  
  Incluye secciones de características, servicios destacados y contacto con mapa embebido.

- **servicios.html**  
  Página secundaria que detalla los servicios ofrecidos por BiciExpress Quito.

- **alquiler.html**  
  Opciones de alquiler de bicicletas para diferentes estilos y niveles.

- **acerca-de.html**  
  Información sobre la empresa y su misión.

- **contacto.html**  
  Datos de contacto, horarios de atención y formulario de consulta.

- **style.css**  
  Archivo de estilos que aplica diseño limpio, navegación horizontal, tipografía clara y estructura visual coherente.

- **banner.jpeg**  
  Imagen utilizada como banner principal en la página de inicio.

---

##  Características destacadas

-  **Amplia Variedad**  
  Bicicletas para todos los estilos y niveles, desde paseos urbanos hasta aventuras de montaña.

-  **Rutas Destacadas**  
  Asesoría sobre las mejores rutas para disfrutar de Quito sobre dos ruedas.

-  **Soporte 24/7**  
  Asistencia técnica y soporte durante todo el recorrido.

---

## Implementación técnica

- Uso de etiquetas semánticas: `<header>`, `<nav>`, `<main>`, `<footer>`.
- Atributo `alt` en imágenes para accesibilidad.
- Enlaces relativos correctamente escritos.
- Diseño adaptable y visualmente funcional.
- Íconos con **Font Awesome** para enriquecer la experiencia visual.
- Mapa embebido de **Google Maps** para mostrar ubicación.
